# <Anchor id="spec" text="商品規格群組管理" />

建立商品規格群組設定。

商品規格群組管理截圖 |
--|
![商品規格群組管理截圖](/winshop-markdown/images/manual/spec/spec_add_group.png) |

### 各號碼說明

1.  **<Anchor id="spec_block_1-1" text="增加規格群組按鈕" ordered sub />**  
    點擊建立或增加規格群組。

    ----

2.  **<Anchor id="spec_block_1-2" text="群組名稱" ordered sub />**  
    設定各個語系的當前群組名稱。

    ----

3.  **<Anchor id="spec_block_1-3" text="啟用開關" ordered sub />**  
    可啟用或停用規格群組。

    ----

4.  **<Anchor id="spec_block_1-4" text="請先儲存按鈕" ordered sub />**  
    建立完規格群組後需要先儲存，再進行下一步的設定。

    ----
    
5.  **<Anchor id="spec_block_1-5" text="儲存按鈕" ordered sub />**  
    點擊可儲存全部的規格群組，包含建立、修改、刪除等的規格群組。

---------------------------------------

點擊「請先儲存」按鈕後畫面截圖 |
--|
![商品規格群組管理儲存後截圖](/winshop-markdown/images/manual/spec/spec_list.png) |

6.  **<Anchor id="spec_block_2-1" text="商品規格項目管理按鈕" ordered sub />**  
    點擊可進入規格項目管理頁面。

---------------------------------------

點擊「商品規格項目管理」按鈕後，進入規格項目管理頁面截圖 |
--|
![商品規格群組管理進入項目頁面截圖](/winshop-markdown/images/manual/spec/spec_add_list.png) |

7.  **<Anchor id="spec_block_3-1" text="增加規格項目按鈕" ordered sub />**  
    點擊建立當前規格群組中的規格項目。

---------------------------------------

新增商品規格項目資料後截圖 |
--|
![設定商品規格項目資料截圖](/winshop-markdown/images/manual/spec/spec_setup_list.png) |

### 各號碼說明

8.  **<Anchor id="spec_block_4-1" text="增加規格項目按鈕" ordered sub />**    
    點擊可在當前規格群組中新增多個規格項目。

    ----

9.  **<Anchor id="spec_block_4-2" text="圖片圖示" ordered sub />**  
    可點擊設定規格項目要顯示的圖片。

    ----

10. **<Anchor id="spec_block_4-3" text="顏色圖示" ordered sub />**  
    可點擊設定規格項目要顯示的背景顏色。

    ----

11. **<Anchor id="spec_block_4-4" text="啟用開關" ordered sub />**  
    可啟用或停用規格項目。

    ----

12. **<Anchor id="spec_block_4-5" text="規格項目名稱" ordered sub />**  
    設定各個語系的當前規格項目名稱。

    ----

13. **<Anchor id="spec_block_4-6" text="取消圖片按鈕" ordered sub />**  
    點擊清除已選擇的圖片圖示。

    ----

14. **<Anchor id="spec_block_4-7" text="取消顏色按鈕" ordered sub />**  
    點擊清除已選擇的顏色圖示。

    ----

15. **<Anchor id="spec_block_4-8" text="刪除按鈕" ordered sub />**  
    點擊刪除整個規格項目。

---------------------------------------

設定完規格項目後截圖 |
--|
![設定完規格項目後截圖](/winshop-markdown/images/manual/spec/spec_save_list.png) |

16.  **<Anchor id="spec_block_5-1" text="儲存按鈕" ordered sub />**    
    點擊可儲存全部的規格項目，包含建立、修改、刪除等的規格項目。
