# 管理員管理

## <Anchor id="admin_add" text="新增管理員" />

新增管理員介面，可自行新增多個管理員，並且賦予權限。

新增管理員表單截圖 |
--|
![新增管理員表單截圖](/images/manual/admin/admin_add.png) |


### 各號碼說明

1.  **<Anchor id="admin_block_1-1" text="設定帳號" ordered sub />**  
    設定管理員的帳號。

    ----

2.  **<Anchor id="admin_block_1-2" text="管理員名稱" ordered sub />**  
    設定新增的管理員名稱。

    ----

3.  **<Anchor id="admin_block_1-3" text="設定信箱" ordered sub />**  
    設定管理員的 E-mail。

    ----

4.  **<Anchor id="admin_block_1-4" text="狀態(啟用切換開關)" ordered sub />**  
    啟用或停用管理員身分。

    ----

5.  **<Anchor id="admin_block_1-5" text="設定密碼" ordered sub />**  
    設定管理員的密碼。

    ----

6.  **<Anchor id="admin_block_1-6" text="確認密碼" ordered sub />**  
    確認管理員的密碼。

    ----

7.  **<Anchor id="admin_block_1-7" text="設定權限群組" ordered sub />**  
    展開下拉選單，選擇已建立的權限群組，即可套用該權限群組的相關權限設定。

    權限群組展開下拉選單 |
    --|
    ![展開權限群組列表](/images/manual/admin/admin_add_permission.png) |

    ----

8.  **<Anchor id="admin_block_1-8" text="刪除已設定的權限群組" ordered sub />**  
    點擊此按鈕，可以刪除選擇的權限群組。

    ----

9.  **<Anchor id="admin_block_1-9" text="增加權限群組" ordered sub />**  
    點擊此按鈕，會增加設定權限群組的下拉欄位，即可設定多組權限。

    可增加多組權限群組 |
    --|
    ![增加多組權限群組](/images/manual/admin/admin_add_addgroup.png) |

----

## <Anchor id="admin_list" text="管理員列表" />

將已有的全部管理員條列顯示。

管理員列表截圖 |
--|
![管理員列表截圖](/images/manual/admin/admin_list.png) |


### 各號碼說明

1.  **<Anchor id="admin_block_2-1" text="帳號" ordered sub />**  
    顯示管理員帳號。

    ----

2.  **<Anchor id="admin_block_2-2" text="名稱" ordered sub />**  
    顯示管理員名稱。

    ----

3.  **<Anchor id="admin_block_2-3" text="管理員信箱" ordered sub />**  
    顯示管理員的 E-mail。

    ----

4.  **<Anchor id="admin_block_2-4" text="權限等級" ordered sub />**  
    顯示該管理員的權限等級。

    ----

5.  **<Anchor id="admin_block_2-5" text="狀態開關" ordered sub />**  
    可於列表中快速啟用或停用管理員。

    ----

6.  **<Anchor id="admin_block_2-6" text="編輯" ordered sub />**  
    點選此按鈕，可進入編輯介面，修改此管理員相關資訊。

    ----

7.  **<Anchor id="admin_block_2-7" text="刪除" ordered sub />**  
    刪除選擇的管理員。

    ----

8.  **<Anchor id="admin_block_2-8" text="搜尋篩選表單開啟" ordered sub>**
    點選此按鈕，會開啟搜尋篩選表單，可按照需求填入「名稱」、「Email」和「等級」來進行資料篩選。
    ![搜尋篩選表單開啟](/images/manual/admin/admin_list_search.png)

----

## <Anchor id="admin_mailbox" text="聯絡信箱" />

可選擇不同的信箱類型，並且分別指定管理員收信。

*  首先選擇信箱類別

聯絡信箱選擇類別截圖 |
--|
![聯絡信箱選擇截圖](/images/manual/admin/admin_mailbox.png) |

*  選擇信箱類別後，即可針對此信箱類別做相關設定。

針對信箱類別設定 |
--|
![信箱類別設定](/images/manual/admin/admin_mailbox_setup.png) |


### 各號碼說明

1.  **<Anchor id="admin_block_3-1" text="信箱類別" ordered sub />**  
    選擇信箱類別，可針對此類別的信箱設定相關的收信資訊。

    ----

2.  **<Anchor id="admin_block_3-2" text="指定管理員" ordered sub />**  
    設定指定的收信管理員，可利用搜尋功能查找已存在的管理員。

    ----

3.  **<Anchor id="admin_block_3-3" text="密件副本開關" ordered sub />**  
    可設定啟用或停用密件副本。

    ----

4.  **<Anchor id="admin_block_3-4" text="刪除" ordered sub />**  
    點選此按鈕，可刪除整個聯絡信箱設定區塊。

    ----

5.  **<Anchor id="admin_block_3-5" text="增加設定區塊" ordered sub />**  
    點選此按鈕，可增加聯絡信箱設定區塊，即可指定不同的收信管理員，並且分別設定是否啟用密件副本。
