# 權限群組管理

## <Anchor id="permission_add" text="新增權限群組" />

網站新增權限群組介面，可自行建立權限群組，提供給相關管理員套用。

新增權限群組表單截圖 |
--|
![新增權限群組截圖上半部](/images/manual/permission/permission_add_top.png) |
![新增權限群組截圖下半部](/images/manual/permission/permission_add_bottom.png) |


### 各號碼說明

1.  **<Anchor id="permission_block_1-1" text="名稱" ordered sub />**  
    設定權限群組的名稱。

    ----

2.  **<Anchor id="permission_block_1-2" text="等級" ordered sub />**  
    設定群組的等級，可依照需求選擇「基本」或是「站台」。
    ![下拉選擇權限群組等級](/images/manual/permission/permission_add_level.png)

    ----

3.  **<Anchor id="permission_block_1-3" text="狀態(啟用切換開關)" ordered sub />**  
    啟用或關閉權限群組。

    ----

4.  **<Anchor id="permission_block_1-4" text="站台權限全開開關" ordered sub />**  
    啟用或關閉站台權限全部開啟的功能。

    ----

5.  **<Anchor id="permission_block_1-5" text="語系權限全開開關" ordered sub />**  
    啟用或關閉語系權限全部開啟的功能。

    ----

6.  **<Anchor id="permission_block_1-6" text="站台權限" ordered sub />**  
    設定群組的所屬站台。

    ----

7.  **<Anchor id="permission_block_1-7" text="語系權限" ordered sub />**  
    設定群組適用的語系。

    ----

8.  **<Anchor id="permission_block_1-8" text="模組權限勾選" ordered sub />**  
    可快速設定群組開啟或關閉最高權限。

    ----

9.  **<Anchor id="permission_block_1-9" text="資料模組權限設定" ordered sub />**  
    可依照需求針對各個資料模組設定權限等級。

    ![下拉設定資料模組權限等級](/images/manual/permission/permission_add_data.png)
     
    *  權限等級說明：

        等級|賦予權限|
        --|--|--|
        LV0：無權限|無權限|
        LV1：讀取權限|僅可讀取|
        LV2：修改權限|可讀取、修改|
        LV3：新增權限|可讀取、修改、新增|
        LV4：刪除權限|可讀取、修改、新增和刪除(最高權限)|

----

## <Anchor id="permission_list" text="權限群組列表" />

將已有的全部權限群組條列顯示。

權限群組列表截圖 |
--|
![權限群組列表截圖](/images/manual/permission/permission_list.png) |


### 各號碼說明

1.  **<Anchor id="permission_block_2-1" text="名稱" ordered sub />**  
    顯示權限群組名稱。

    ----

2.  **<Anchor id="permission_block_2-2" text="等級" ordered sub />**  
    顯示群組的權限等級。

    ----

3.  **<Anchor id="permission_block_2-3" text="狀態開關" ordered sub />**  
    可於列表中快速啟用或停用群組。

    ----

4.  **<Anchor id="permission_block_2-4" text="編輯" ordered sub />**  
    點選此按鈕，可進入編輯介面，針對此權限群組修改相關設定。

    ----

5.  **<Anchor id="permission_block_2-5" text="刪除" ordered sub />**  
    刪除選擇的權限群組。

    ----

6.  **<Anchor id="permission_block_2-6" text="搜尋篩選表單開啟" ordered sub />**  
    點選此按鈕，會開啟搜尋篩選表單，可按照需求填入「名稱」、「等級」和「所屬站台」來進行資料篩選。
    ![搜尋篩選表單開啟](/images/manual/permission/permission_list_search.png)
