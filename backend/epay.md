# 金流管理

## 金流設定

建立金流設定。

金流設定截圖 |
--|
![金流設定截圖](/winshop-markdown/images/manual/epay/epay_setup.png) |


### 各號碼說明

1.  **<h2>選擇金流系統</h2>**  
    開啟下拉選單，選擇金流系統。

    開啟金流系統選單 |
    --|
    ![開啟金流系統選單](/winshop-markdown/images/manual/epay/epay_setup_system.png) |

    ----

2.  **<h2>刪除</h2>**  
    點擊此按鈕，將刪除整個金流設定區塊。

    ----

3.  **<h2>增加</h2>**  
    點擊此按鈕，可以增加整個金流設定區塊。

    ----

4.  **<h2>付款管理</h2>**  
    點擊此按鈕，會跳轉到付款管理頁面。[按此前往付款管理頁面](#epay_pay)

----

## 付款管理

建立並設定付款方式。

付款管理截圖 |
--|
![付款管理截圖](/winshop-markdown/images/manual/epay/epay_pay.png) |


### 各號碼說明

1.  **<h2>增加付款選項</h2>**  
    點擊此按鈕，可以增加自訂付款方式的區塊。

    ----

2.  **<h2>自訂付款方式名稱</h2>**  
    設定付款方式的名稱。

    ----

3.  **<h2>刪除</h2>**  
    點擊此按鈕，將刪除整個自訂付款方式區塊。

    ----

4.  **<h2>啟用開關</h2>**  
    設定啟用或停用此付款方式。
