# 會員管理

## <Anchor id="users_add" text="新增會員" />

建立網站會員，並設定相關基本資料。

1.  **<Anchor id="users_block_1-1" text="帳號" ordered sub />**  
    設定會員帳號。

    ----

2.  **<Anchor id="users_block_1-2" text="密碼" ordered sub />**  
    設定會員密碼。

    ----

3.  **<Anchor id="users_block_1-3" text="密碼確認" ordered sub />**  
    確認設定的會員密碼。

    ----

4.  **<Anchor id="users_block_1-4" text="姓名" ordered sub />**  
    設定會員姓名。

    ----

5.  **<Anchor id="users_block_1-5" text="性別" ordered sub />**  
    選擇會員性別。

    ----

6.  **<Anchor id="users_block_1-6" text="電話" ordered sub />**  
    設定會員電話。

    ----

7.  **<Anchor id="users_block_1-7" text="手機" ordered sub />**  
    設定會員手機。

    ----

8.  **<Anchor id="users_block_1-8" text="Email" ordered sub />**  
    設定會員信箱。

    ----

9.  **<Anchor id="users_block_1-9" text="生日" ordered sub />**  
    設定會員生日。

    開啟日期選擇器 |

    設定會員地址的區域。

    展開下拉選單選擇區域 |
    --|
    ![下拉選單選擇](/images/manual/users/users_add_region.png) |

    ----

13.  **<Anchor id="users_block_1-13" text="地址" ordered sub />**  
    設定會員地址的地址。

----

## <Anchor id="users_list" text="會員列表" />

將已有的全部會員列表顯示。

會員列表截圖 |
--|
![會員列表截圖](/images/manual/users/users_list.png) |


### 各號碼說明

1.  **<Anchor id="users_block_2-1" text="新增會員按鈕" ordered sub />**  
    點擊此按鈕，即跳轉到新增會員頁面，可新增會員。

    ----

2.  **<Anchor id="users_block_2-2" text="全選按鈕" ordered sub />**  
    點擊此按鈕，可快速選取全部會員項目，或是全部不選取。

    點擊選取全部會員 |
    --|
    ![選取全部會員](/images/manual/users/users_list_select.png) |

    ----

3.  **<Anchor id="users_block_2-3" text="匯出Excel按鈕" ordered sub />**  
    點擊此按鈕，會取得已選取的會員資料，並且產生一個 Excel 檔案以供下載。

    取得資料後產生 Excel 檔案 |
    --|
    ![產生Excel檔案](/images/manual/users/users_list_export.png) |

    ----

4.  **<Anchor id="users_block_2-4" text="開啟搜尋條件表單按鈕" ordered sub />**  
    點擊此按鈕，將搜尋條件表單展開。可依照「帳號是否啟用」、「是否為VIP會員」、「註冊時間範圍」和關鍵字等條件來進行搜尋會員。

    開啟搜尋條件表單 |
    --|
    ![開啟搜尋條件表單](/images/manual/users/users_list_search.png) |

    ----

5.  **<Anchor id="users_block_2-5" text="欄位選項按鈕" ordered sub />**  
    點擊此按鈕，開啟欄位選項下拉選單，可自行選擇要顯示在列表的會員資料欄位。

    選擇要顯示的欄位選項 |
    --|
    ![開啟欄位選項下拉選單](/images/manual/users/users_list_item.png) |

    ----

6.  **<Anchor id="users_block_2-6" text="會員資料列表" ordered sub />**  
    條列式顯示會員資料。「狀態」和「帳號啟用」欄位開關可以快速切換啟用或停用狀態；點擊「編輯」按鈕可以進入會員資料編輯介面；點擊「刪除」按鈕能夠刪除所選會員。

    會員列表快速按鈕 |
    --|
    ![會員列表快速按鈕](/images/manual/users/users_list_strip.png) |

----

## <Anchor id="users_import" text="會員匯入" />

可以將會員資料檔案（Excel檔）匯入。點擊「請選擇」按鈕，可以從檔案總管中開啟 Excel 檔案，或是上傳 Excel 檔案。
