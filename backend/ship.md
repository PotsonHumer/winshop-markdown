# 運費管理

## <Anchor id="ship_add" text="新增運費" />

新增運費，並設定相關資訊。

運費管理截圖 |
--|
![新增運費截圖](/winshop-markdown/images/manual/ship/ship_add.png) |

### 各號碼說明

1.  **<Anchor id="ship_block_1-1" text="運費名稱" ordered sub />**  
    輸入運費名稱，在選擇運費時顯示。

    ----

2.  **<Anchor id="ship_block_1-2" text="啟用開關" ordered sub />**  
    可啟用或停用運費。

    ----

3.  **<Anchor id="ship_block_1-3" text="增加物流搭配按鈕" ordered sub />**  
    點擊可新增物流搭配，選取已建立的物流系統及物流方法。

    點擊按鈕後，新增一筆物流搭配。各號碼說明如下|
    --|
    ![新增物流搭配截圖]](/winshop-markdown/images/manual/ship/ship_add_logistics.png) |    

    各號碼說明：

    欄位名稱|欄位說明 |
    --|--|
    1.物流系統|下拉選擇已建立的物流系統 |
    2.物流方法|下拉選擇已建立的物流方法 |
    3.刪除按鈕|點擊可刪除該筆物流搭配 |
    4.增加物流搭配按鈕|可增加多筆物流搭配 |

    ----

4.  **<Anchor id="ship_block_1-4" text="增加費用設定按鈕" ordered sub />**  
    點擊可新增費用設定，選取已建立的運送規格，並設定比對數值、運費金額和免運價格門檻等資訊。

    點擊按鈕後，新增一筆運費費用。各號碼說明如下|
    --|
    ![新增運費費用截圖]](/winshop-markdown/images/manual/ship/ship_add_price.png) |    

    各號碼說明：

    欄位名稱|欄位說明 |
    --|--|
    1.運送規格|下拉選擇已建立的運送規格 |
    2.比對數值|依照所選擇的運送規格決定單位，設定比對數值（商品總規格數值大於比對數值則套用這筆運費） |
    3.運費|設定運費費用 |
    4.免運價格門檻|設定可決定免運的價格門檻（設定為 0 則表示免運） |
    5.無免運費| 勾選則設定為免運（免運價格門檻欄位自動清除）|
    6.刪除按鈕| 點擊可刪除該筆運費費用 |
    7.增加費用設定按鈕|可增加多筆運費費用 |

----

## <Anchor id="ship_list" text="運費管理" />

將所有的運費資料列出以供管理。

運費管理截圖 |
--|
![運費管理截圖](/winshop-markdown/images/manual/ship/ship_list.png) |

### 各號碼說明

1.  **<Anchor id="ship_block_2-1" text="新增運費資料" ordered sub />**  
    點擊新增運費資料。

    ----

2.  **<Anchor id="ship_block_2-2" text="啟用開關" ordered sub />**  
    可啟用或停用運費資料。

    ----

3.  **<Anchor id="ship_block_2-3" text="編輯按鈕" ordered sub />**  
    點擊進入運費更新頁面，可編輯運費相關設定。

    點擊按鈕後，跳轉到更新運費資料頁面|
    --|
    ![更新運費資料截圖]](/winshop-markdown/images/manual/ship/ship_update.png) | 

    ----

4.  **<Anchor id="ship_block_2-4" text="刪除按鈕" ordered sub />**  
    點擊可刪除所選的運費資料。

----

## <Anchor id="ship_spec" text="商品運送規格" />

運費管理截圖 |
--|
![運費管理截圖](/winshop-markdown/images/manual/ship/ship_spec.png) |

### 各號碼說明

1.  **<Anchor id="ship_block_3-1" text="增加商品運送規格按鈕" ordered sub />**  
    點擊可增加商品運送規格，設定規格名稱、單位，並決定是否累計計算。

    ----

2.  **<Anchor id="ship_block_3-2" text="規格名稱" ordered sub />**  
    設定商品運送規格的名稱。

    ----

3.  **<Anchor id="ship_block_3-3" text="規格單位" ordered sub />**  
    設定商品運送規格的單位。

    ----

4.  **<Anchor id="ship_block_3-4" text="累計計算運費" ordered sub />**  
    勾選則設定為累計計算運費（商品總規格數值每大於一組比對數值則計算一次運費，最後全部加總為運費費用）。

    ----

5.  **<Anchor id="ship_block_3-5" text="刪除按鈕" ordered sub />**  
    點擊可刪除該筆商品運送規格。
