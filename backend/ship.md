# 運費管理

## 新增運費

新增運費，並設定相關資訊。

| 運費管理截圖 |
|---|
| ![新增運費截圖](/winshop-markdown/images/manual/ship/ship_add.png) |

### 各號碼說明

1.  **運費名稱**  
    輸入運費名稱，在選擇運費時顯示。

    ----

2.  **啟用開關**  
    可啟用或停用運費。

    ----

3.  **增加物流搭配按鈕**  
    點擊可新增物流搭配，選取已建立的物流系統及物流方法。

    | 點擊按鈕後，新增一筆物流搭配。各號碼說明如下 |
    |---|
    | ![新增物流搭配截圖](/winshop-markdown/images/manual/ship/ship_add_logistics.png) |

    各號碼說明：

    | 欄位名稱 | 欄位說明 |
    |---|---|
    | 1.物流系統 | 下拉選擇已建立的物流系統 |
    | 2.物流方法 | 下拉選擇已建立的物流方法 |
    | 3.刪除按鈕 | 點擊可刪除該筆物流搭配 |
    | 4.增加物流搭配按鈕 | 可增加多筆物流搭配 |

    ----

4.  **增加費用設定按鈕**  
    點擊可新增費用設定，選取已建立的運送規格，並設定比對數值、運費金額和免運價格門檻等資訊。

    | 點擊按鈕後，新增一筆運費費用。各號碼說明如下 |
    |---|
    | ![新增運費費用截圖](/winshop-markdown/images/manual/ship/ship_add_price.png) |

    各號碼說明：

    | 欄位名稱 | 欄位說明 |
    |---|---|
    | 1.運送規格 | 下拉選擇已建立的運送規格 |
    | 2.比對數值 | 依照所選擇的運送規格決定單位，設定比對數值（商品總規格數值大於比對數值則套用這筆運費） |
    | 3.運費 | 設定運費費用 |
    | 4.免運價格門檻 | 設定可決定免運的價格門檻（設定為 0 則表示免運） |
    | 5.無免運費 | 勾選則設定為免運（免運價格門檻欄位自動清除） |
    | 6.刪除按鈕 | 點擊可刪除該筆運費費用 |
    | 7.增加費用設定按鈕 | 可增加多筆運費費用 |

----

## 運費管理

將所有的運費資料列出以供管理。

| 運費管理截圖 |
|---|
| ![運費管理截圖](/winshop-markdown/images/manual/ship/ship_list.png) |

### 各號碼說明

1.  **新增運費資料**  
    點擊新增運費資料。

    ----

2.  **啟用開關**  
    可啟用或停用運費資料。

    ----

3.  **編輯按鈕**  
    點擊進入運費更新頁面，可編輯運費相關設定。

    | 點擊按鈕後，跳轉到更新運費資料頁面 |
    |---|
    | ![更新運費資料截圖](/winshop-markdown/images/manual/ship/ship_update.png) |

    ----

4.  **刪除按鈕**  
    點擊可刪除所選的運費資料。

----

## 商品運送規格

| 運費管理截圖 |
|---|
| ![運費管理截圖](/winshop-markdown/images/manual/ship/ship_spec.png) |

### 各號碼說明

1.  **增加商品運送規格按鈕**  
    點擊可增加商品運送規格，設定規格名稱、單位，並決定是否累計計算。

    ----

2.  **規格名稱**  
    設定商品運送規格的名稱。

    ----

3.  **規格單位**  
    設定商品運送規格的單位。

    ----

4.  **累計計算運費**  
    勾選則設定為累計計算運費（商品總規格數值每大於一組比對數值則計算一次運費，最後全部加總為運費費用）。

    ----

5.  **刪除按鈕**  
    點擊可刪除該筆商品運送規格。
