# 聯絡我們管理

聯絡我們資料條列式顯示，能夠查看詳細內容，也可以刪除資料。

聯絡我們管理截圖 |
--|
![聯絡我們管理截圖](/winshop-markdown/images/manual/contactus/contactus.png) |


### 各號碼說明

1.  **開啟搜尋條件表單**  
    點擊此按鈕，將搜尋條件表單展開。可依照關鍵字、來源IP、「搜尋起始訂單時間」和「搜尋結束訂單時間」等條件來進行搜尋資料。

    開啟搜尋條件表單 |
    --|
    ![開啟搜尋條件表單](/winshop-markdown/images/manual/contactus/contactus_search.png) |

    ----

2.  **欄位選項**  
    點擊此按鈕，開啟欄位選項下拉選單，可自行選擇要顯示在列表的資料欄位。

    選擇要顯示的欄位選項 |
    --|
    ![開啟欄位選項下拉選單](/winshop-markdown/images/manual/contactus/contactus_item.png) |

    ----

3.  **聯絡我們資料列表**  
    條列式顯示聯絡我們資料。點擊「詳細」按鈕可以進入聯絡我們詳細管理頁面；點擊「刪除」按鈕能夠刪除所選聯絡我們資料。

    聯絡我們資料列表快速按鈕 |
    --|
    ![聯絡我們資料列表快速按鈕](/winshop-markdown/images/manual/contactus/contactus_strip.png) |

    ----

4.  **群組刪除**  
    選取多筆聯絡我們資料後點擊此按鈕，可以一次刪除勾選的所有資料。
