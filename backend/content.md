# 內容管理

## 內容列表

樹狀圖最後一層為資料，其餘皆為分類。<small>(見下圖：以商品管理為例)</small>

| 內容列表樹狀截圖 |
|---|
| ![內容列表樹狀截圖](/winshop-markdown/images/manual/content/content_list.png) |


### 各號碼說明

1.  **搜尋資料**  
    輸入資料關鍵字，可快速查找資料。

    ----

2.  **新增子項目**  
    點擊此按鈕，可建立該分類層底下的子項目。

    | 點擊按鈕後，會在該分類下層列表的最後增加一個「新項目」。可點擊編輯按鈕進入編輯介面，或是點擊刪除按鈕刪除項目。 |
    |---|
    | ![新增的子項目位於列表最後](/winshop-markdown/images/manual/content/content_list_add.png) |

    ----

3.  **編輯**  
    點擊此按鈕，可進入編輯介面，針對此筆資料進行修改。[按此前往說明](#content_edit)

    ----

4.  **列表icon**  
    按住列表icon進行拖曳，可調整排列順序或者變更分類。

    | 拖曳時底色為藍色，則可移動 | 拖曳時底色為紅色，代表不可移動至此位置 |
    |---|---|
    | ![拖曳項目可移動](/winshop-markdown/images/manual/content/content_list_drag.png) | ![拖曳項目不可移動](/winshop-markdown/images/manual/content/content_list_notdrag.png) |

    ----

5.  **刪除**  
    刪除選擇的內容項目。

    ----

6.  **建立跨分類連結**  
    點擊此按鈕，會建立一筆資料內容完全相同並且互相連動的資料。建立的位置預設為當前分類，可以自行拖曳至想放置的位置。    

    ----

7.  **顯示是否啟用**  
    此區塊可顯示此產品是否啟用；如果啟用，顯示為綠色打勾，如果為停用，則顯示紅色禁止 icon。

    | 啟用為綠色打勾，停用為紅色禁止 icon |
    |---|
    | ![顯示是否啟用](/winshop-markdown/images/manual/content/content_list_status.png) |

    ----

8.  **庫存數量**  
    顯示此商品的庫存數量。<small>(僅限商品內容，其餘內容資料無此欄位)</small>

    | 進入商品編輯介面，可自行設定庫存數量 |
    |---|
    | ![編輯介面中的庫存欄位](/winshop-markdown/images/manual/content/content_edit_stock.png) |

    ----

9.  **取消跨分類連結**  
    點擊此按鈕，會將藉由跨分類連結產出的當前資料刪除，原本的那筆資料不會受到影響。

    ----

10.  **最新標籤**  
    如果項目設定為最新標記，則會顯示此標籤。

    | 進入編輯介面，可啟用或停用最新標記 |
    |---|
    | ![編輯介面中的最新標記啟用開關](/winshop-markdown/images/manual/content/content_edit_latest.png) |

    ----

11.  **熱門標籤**  
    如果項目設定為熱門標記，則會顯示此標籤。

    | 進入編輯介面，可啟用或停用熱門標記 |
    |---|
    | ![編輯介面中的熱門標記啟用開關](/winshop-markdown/images/manual/content/content_edit_hot.png) |

    ----

12.  **促銷標籤**  
    如果項目設定為促銷標記，則會顯示此標籤。<small>(僅限商品內容，其餘內容資料無此欄位)</small>

    | 進入商品編輯介面，可啟用或停用促銷標記 |
    |---|
    | ![編輯介面中的促銷標記啟用開關](/winshop-markdown/images/manual/content/content_edit_sale.png) |

    ----

13.  **商品規格搭配管理按鈕**  
    點擊此按鈕，可進入商品規格搭配管理介面。[按此前往說明](#content_spec)

----

## 編輯介面

進入編輯介面，即可對項目資料進行修改。

| 分別說明各個頁籤 |
|---|
| ![編輯介面的頁籤](/winshop-markdown/images/manual/content/content_edit_tab.png) |

### 各號碼說明

1. **基本設定**  
    項目資料的基本資料設定。

    | 基本資料頁籤截圖 |
    |---|
    | ![編輯介面的基本設定頁籤上半部](/winshop-markdown/images/manual/content/content_edit_setup_top.png) |
    | ![編輯介面的基本設定頁籤下半部](/winshop-markdown/images/manual/content/content_edit_setup_bottom.png) |

    各號碼說明：<small>(型號、價格、折價、運費、庫存、詢價商品開關、促銷標記開關僅限商品內容，其餘內容資料沒有這些欄位)</small>

    | 欄位名稱 | 欄位說明 |
    |---|---|
    | 1.標題 | 輸入項目名稱 |
    | 2.型號 | 輸入商品型號 |
    | 3.價格 | 設定商品價格 |
    | 4.折價 | 設定商品折價數；輸入數字為金額折扣，輸入小數點則為打折數（例如：0.8 為打八折） |
    | 5.庫存 | 設定商品庫存數量 |
    | 6.網址 | 項目自定義的網址 |
    | 7.重新導向網址 | 可自行設定導向網址，則會跳轉到該網址 |
    | 8.簡述 | 輸入項目的簡短說明 |
    | 9.狀態開關 | 啟用或關閉項目|
    | 10.詢價商品開關 | 停用此功能，前台該商品可直接加入購物車；啟用此功能，則前台該商品不可直接加入購物車，顯示為加入詢價單 |
    | 11.最新標記開關 | 可啟用或停用最新標記 |
    | 12.熱門標記開關 | 可啟用或停用熱門標記 |
    | 13.促銷標記開關 | 可啟用或停用促銷標記 |
    | 14.商品規格搭配管理按鈕 | 點擊按鈕跳轉到商品規格搭配介面，可以設定商品規格相關資料。[按此前往商品規格搭配管理介面說明](#content_edit) |
    | 15.複製按鈕 | 點擊按鈕開啟其他語系，選擇想要複製的語系（複製所有頁籤資料到當前語系） |

    最新消息管理模組：

    | 額外欄位 | 欄位說明 |
    |---|---|
    | 顯示時間 | 設定在前台顯示的時間 |
    | 啟用時間 | 設定啟用的時間 |

    ----

2. **SEO 設定**  
    項目資料的 SEO 相關資料設定。

    | SEO 設定頁籤截圖 |
    |---|
    | ![編輯介面的 SEO 設定頁籤](/winshop-markdown/images/manual/content/content_edit_seo.png) |

    各號碼說明：

    | 欄位名稱 | 欄位說明 |
    |---|---|
    | 1.網頁標題 | 設定資料網頁的標題 |
    | 2.關鍵字 | 設定資料網頁的關鍵字 |
    | 3.收錄開關 | 設定資料網頁收錄與否 |
    | 4.描述 | 設定資料網頁的描述 |

    ----

3. **運費相關設定**
    項目資料的運費相關設定。

    | 運費相關設定頁籤截圖 |
    |---|
    | ![編輯介面的運費相關設定頁籤](/winshop-markdown/images/manual/content/content_edit_ship.png) |

    各號碼說明：

    | 欄位名稱 | 欄位說明 |
    |---|---|
    | 1.排除運費項目 | 開啟下拉選單，選取要排除的運費項目 |
    | 2.運費規格數值 | 設定運費規格數值（公斤） |

    ----

4. **圖片設定**  
    項目資料的圖片，可加入多張圖片，並分別設定相關資訊。

    | 設定圖片頁籤截圖 |
    |---|
    | ![編輯介面的設定圖片頁籤](/winshop-markdown/images/manual/content/content_edit_image.png) |

    各號碼說明：

    | 欄位名稱 | 欄位說明 |
    |---|---|
    | 1.圖片 | 選擇資料圖片 |
    | 2.TITLE | 設定此圖片的 title（滑鼠移至圖片會顯示的文字） |
    | 3.ALT | 設定此圖片的 alt（圖片的替代文字） |
    | 4.刪除按鈕 | 刪除整個圖片設定區塊 |
    | 5.增加圖片按鈕 | 增加整個圖片設定區塊 |

    例外的資料模組：

    | 資料模組 | 例外狀況 |
    |---|---|
    | 關於我們管理 | 僅可放一張首頁圖片、一張視差滾動背景圖（區塊固定） |
    | 最新消息管理、檔案下載管理 | 僅可放一張圖片作為封面圖片 |
    | 常見問題管理、單頁訊息管理 | 沒有設定圖片頁籤 |

    ----

5. **附件設定**  
    增加項目資料的附件。

    | 設定附件頁籤截圖 |
    |---|
    | ![編輯介面的設定附件頁籤](/winshop-markdown/images/manual/content/content_edit_attach.png) |

    各號碼說明：

    | 欄位名稱 | 欄位說明 |
    |---|---|
    | 1.檔案名稱 | 原始檔案的名稱 |
    | 2.標題 | 可自行設定附件顯示的名稱，如果為空則顯示檔案名稱 |
    | 3.發布日期 | 可自行設定附件的發布日期，建立附件時預設為當天日期 |
    | 4.變更附件按鈕 | 點擊按鈕即開啟檔案總管，可快速更換附件檔案 |
    | 5.刪除按鈕 | 刪除整個附件設定區塊 |
    | 6.增加附件按鈕 | 增加整個附件設定區塊 |

    ----

6. **描述設定**  
    增加項目資料的描述（內文）。

    | 設定描述頁籤截圖 |
    |---|
    | ![編輯介面的設定描述頁籤](/winshop-markdown/images/manual/content/content_edit_description.png) |

    各號碼說明：

    | 欄位名稱 | 欄位說明 |
    |---|---|
    | 1.描述抬頭 | 顯示在每個描述區塊的標題，預設為 DescriptionName，可自行修改。（商品內容中是必填欄位，顯示為商品內頁中的標籤名稱） |
    | 2.編輯器內容 | 描述的內容，可利用編輯器做各種排版 |
    | 3.刪除按鈕 | 刪除整個描述設定區塊 |
    | 4.增加按鈕 | 增加整個描述設定區塊 |

    ----

7. **關聯商品**  
    設定此商品的關聯商品。(僅限商品內容，其餘內容資料無此頁籤)

    | 關聯商品頁籤截圖 |
    |---|
    | ![編輯介面的關聯商品頁籤](/winshop-markdown/images/manual/content/content_edit_relate.png) |

    各號碼說明：

    | 欄位名稱 | 欄位說明 |
    |---|---|
    | 1.目前商品 | 標示出目前的商品資料 |
    | 2.取消關聯商品按鈕 | 代表此商品已設定為關聯商品，點擊按鈕即可取消關聯 |
    | 3.關聯商品按鈕 | 代表此商品不是關聯商品，可點擊按鈕建立關聯 |

----

## 商品規格搭配管理介面

點擊商品編輯頁面中的「商品規格搭配管理」按鈕，可以跳轉到商品規格搭配管理介面，設定商品規格的相關搭配。  
<Icon icon="fa fa-exclamation-triangle" text="僅針對此商品做規格搭配設定，不會影響到共用的商品規格群組。" color="warning" />  

| 分別說明各個頁籤 |
|---|
| ![商品規格搭配管理介面的頁籤](/winshop-markdown/images/manual/content/content_spec_tab.png) |

### 各號碼說明

1. **啟用規格項目設定**  
    需要先選取要進行規格搭配的規格群組。

    | 啟用規格項目設定頁籤截圖 |
    |---|
    | ![商品規格搭配管理介面的啟用規格項目設定頁籤](/winshop-markdown/images/manual/content/content_spec_enable.png) |

    各號碼說明：

    | 欄位名稱 | 欄位說明 |
    |---|---|
    | 1.選擇商品規格群組 | 開啟下拉選單，選擇已建立的商品規格群組 |
    | 2.商品規格群組列表 | 將已選取的全部商品規格群組展開列出來（顯示每個群組中所有規格項目） |
    | 3.移動 icon | 按住移動 icon 或是規格項目，即可進行拖曳，調整規格項目順序 |
    | 4.編輯按鈕 | 點擊按鈕後，會開啟規格項目表單視窗，可針對該筆商品規格項目進行相關設定（欄位說明可參考：商品規格群組管理中的商品規格項目管理介面） |
    | 5.刪除按鈕 | 點擊按鈕後，可以刪除該筆商品規格項目 |
    | 6.增加規格項目按鈕 | 點擊按鈕後，在該商品規格群組中增加規格項目 |
    | 7.儲存按鈕 | 需要先儲存，才能接著建立規格搭配 |
    | 8.回到商品設定按鈕 | 點擊按鈕後，可以回到商品設定介面 |
    
    ----

2. **建立規格搭配**  
    將所選的商品規格項目進行搭配。

    | 建立規格搭配頁籤截圖 |
    |---|
    | ![商品規格搭配管理介面的建立規格搭配頁籤](/winshop-markdown/images/manual/content/content_spec_match.png) |

    各號碼說明：

    | 欄位名稱 | 欄位說明 |
    |---|---|
    | 1.選擇商品規格項目 | 開啟下拉選單，從已選擇的規格群組中選擇規格項目 |
    | 2.全選按鈕 | 點擊按鈕後，取得已選擇的規格群組中的全部規格項目 |
    | 3.商品規格項目搭配列表 | 將已選取的規格項目搭配後列出來 |
    | 4.顯示圖片 | 點擊圖片區塊選擇圖片，在前台選擇此規格項目時顯示 |
    | 5.啟用開關 | 可啟用或停用此筆規格搭配項目 |
    | 6.規格型號 | 可設定此筆規格搭配項目的型號，在前台選擇此規格項目時顯示 |
    | 7.規格價格 | 可設定此筆規格搭配項目的價格，在前台選擇此規格項目時顯示 |
    | 8.庫存數量 | 可設定此筆規格搭配項目的庫存數量，在前台選擇此規格項目時顯示 |
    | 9.取消圖片按鈕 | 點擊清除已選擇的圖片圖示 |
    | 10.刪除按鈕 | 點擊刪除此筆規格搭配項目 |

    ----

3. **規格搭配設定**  
    可以針對已建立的規格搭配進行設定、修改、刪除。

    | 規格搭配設定頁籤截圖 |
    |---|
    | ![商品規格搭配管理介面的規格搭配設定頁籤](/winshop-markdown/images/manual/content/content_spec_setup.png) |

    將已建立並存檔的規格搭配項目列出來，並可以針對每一筆規格搭配進行修改和刪除。
