# 橫幅廣告管理

可依照不同資料模組，分別建立一或多個橫幅廣告。

*  首先選擇模組

| 橫幅廣告模組截圖 |
|---|
| ![橫幅廣告模組截圖](/winshop-markdown/images/manual/banner/banner.png) |

*  選擇模組後，即可針對此模組設定橫幅廣告。

| 針對模組設定橫幅 |
|---|
| ![針對模組設定橫幅](/winshop-markdown/images/manual/banner/banner_list.png) |

### 各號碼說明

1.  **建立橫幅廣告**

    點擊此按鈕，進入建立橫幅廣告介面。

    | 建立橫幅廣告介面 |
    |---|
    | ![建立橫幅廣告介面](/winshop-markdown/images/manual/banner/banner_edit.png) |

    各號碼說明：

    | 欄位名稱 | 欄位說明 |
    |---|---|
    | 1.廣告圖片 | 選擇橫幅廣告的圖片 |
    | 2.TITLE|可設定橫幅廣告的抬頭，結果會顯示在橫幅區塊 |
    | 3.ALT|可設定此圖片的 alt（圖片的替代文字） |
    | 4.連結網址|可設定橫幅廣告的連結網址 |
    | 5.簡述|可設定橫幅廣告的簡述文字 |
    | 6.啟用開關|可啟用或停用橫幅廣告 |

    | 建立橫幅廣告介面 |
    |---|
    | ![建立橫幅廣告介面](/winshop-markdown/images/manual/banner/banner_video_top.png) |
    | ![建立橫幅廣告介面](/winshop-markdown/images/manual/banner/banner_video_bottom.png) |

    各號碼說明：

    **圖片和影片只能擇一設定。**
    
    | 欄位名稱 | 欄位說明 |
    |---|---|
    | 1.廣告圖片 | 選擇橫幅廣告的圖片 |
    | 2.TITLE|可設定橫幅廣告的抬頭，結果會顯示在橫幅區塊 |
    | 3.ALT|可設定此圖片的 alt（圖片的替代文字） |
    | 4.廣告影片|已選擇的橫幅廣告影片 |
    | 5.選擇影片按鈕|點擊可開啟檔案總管，選擇要設定的橫幅廣告影片 |
    | 6.連結網址|可設定橫幅廣告的連結網址 |
    | 7.簡述|可設定橫幅廣告的簡述文字 |
    | 8.啟用開關|可啟用或停用橫幅廣告 |
    

    ----

2.  **模組名稱**  
    顯示所選的資料模組名稱（當前所在）。

    ----

3.  **橫幅廣告抬頭**  
    顯示設定的橫幅 TITLE，在橫幅內容的 TITLE 欄位設定（[設定畫面截圖 ─ 號碼2](#banner_block_1-1)）。

    ----

4.  **廣告圖片**  
    顯示設定的廣告圖片。

    顯示設定的廣告圖片。（如果選擇影片則會顯示廣告影片）

    ----

5.  **啟用開關**  
    可啟用或停用此橫幅廣告。

    ----

6.  **更新橫幅廣告**  
    點擊此按鈕，會進入更新橫幅廣告頁面，可修改橫幅的各項設定。

    ----
    
7.  **刪除**  
    點擊此按鈕，刪除整個橫幅廣告區塊。
